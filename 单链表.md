# 一.单链表遍历的核心原理

1. **起点**：定义一个指针 `cur`，让它指向链表的**第一个节点**（即头指针 `head` 所指向的节点）。
2. **访问当前节点**：通过 `cur` 指针访问数据（如 `cur->val`）。
3. **为什么有了表头指针还要创建 `cur` 指针来遍历？** 核心原因：不能移动 `head` 本身，否则会丢失链表入口。
4. **移动到下一个节点**：将 `cur` 更新为 `cur->next`。
5. **重复**：继续访问新节点的数据，然后移动，直到 `cur` 变成 `NULL`。

```C
LTNode* cur = head;
while (cur != NULL) {
    printf("%d ", cur->val);
    cur = cur->next;
}
```

> **遍历链表就是用一个指针沿着 `next` 链从第一个节点走到 `NULL`，逐个访问每个节点。**

# 二.区分节点和指针域（next）

### 1. 什么是节点？
一个节点是一个结构体，包含两部分：
- **数据域**（`a1`, `a2`...）
- **指针域**（`next`）

`a1` 连同它后面的 `next` 合在一起才构成一个完整的节点。

### 2. `next` 是什么？
`next` 是节点里的一个成员变量，类型是指针，**存储的是下一个节点的地址**。它本身**不是节点**，而是节点内部的一个"箭头"。

# 三.指向某个节点具体是指在哪里

## 1、内存视角
当节点被 `malloc` 分配在堆上时，它会有一个**起始地址**，比如 `0x1000`。

```
内存地址      内容
0x1000        [ val = 10 ]       // 数据域
0x1008        [ next = 0x2000 ]  // 指针域
```

"指向第一个节点" = 变量 `head` 里存的数值是 `0x1000`。

## 2、代码视角

```C
LTNode* head = malloc(sizeof(LTNode));
head->val = 10;
head->next = NULL;
```

`head` 是一个指针变量，里面存的是第一个节点的**起始地址**。

# 四.为什么要用malloc动态分配内存

链表**节点个数不确定、需要在运行时动态增减、且每个节点的生命周期需要跨函数存在**。

### 局部变量（栈上）的问题
函数结束后栈上节点被销毁，返回地址变成悬空指针。

### 全局数组（静态区）的问题
最大节点数固定，无法动态扩展。

### malloc 的好处
- **按需分配**：需要几个就分配几个
- **生命周期手动控制**：直到主动 `free` 才释放
- **灵活删除**：可以 `free` 任意节点

# 五.完整演示遍历链表操作

```C
#include <stdio.h>
#include <stdlib.h>

typedef struct ListNode {
    int val;
    struct ListNode* next;
} LTNode;

int main() {
    LTNode* head = NULL;

    LTNode* node1 = (LTNode*)malloc(sizeof(LTNode));
    node1->val = 10; node1->next = NULL;
    head = node1;

    LTNode* node2 = (LTNode*)malloc(sizeof(LTNode));
    node2->val = 20; node2->next = NULL;
    node1->next = node2;

    LTNode* node3 = (LTNode*)malloc(sizeof(LTNode));
    node3->val = 30; node3->next = NULL;
    node2->next = node3;

    // 遍历
    LTNode* cur = head;
    while (cur != NULL) {
        printf("%d ", cur->val);
        cur = cur->next;
    }
    return 0;
}
```

> 链表存储结构中，指针首先指向有效值，"若无有效值直接指向next"。当链表为空时，头指针 `head` 直接指向 NULL。**链表的本质就是指针的递归**。

```C
#include <iostream>
#include <list>
int main() {
    std::list<int> lst = {10, 20, 30};
    for (int val : lst) std::cout << val << " ";
}
```

```C
struct Node {
    int val;
    Node* next;
    Node(int x) : val(x), next(nullptr) {}
};
int main() {
    Node* head = new Node(10);
    head->next = new Node(20);
    head->next->next = new Node(30);
    for (Node* cur = head; cur; cur = cur->next)
        std::cout << cur->val << " ";
}
```
