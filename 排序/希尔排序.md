# 希尔排序

希尔排序分为两步：
- **预排序**：让数组接近有序
- **插入排序**

```C
void shellSort(int arr[], int n) {
    for(int gap = n/2; gap > 0; gap /= 2) {
        for(int start = 0; start < gap; start++) {
            for(int i = start + gap; i < n; i += gap) {
                int tmp = arr[i];
                int j = i - gap;
                while(j >= start && tmp < arr[j]) {
                    arr[j + gap] = arr[j];
                    j -= gap;
                }
                arr[j + gap] = tmp; 
            }
        }
    }
}
```
