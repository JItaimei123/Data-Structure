# 插入排序

```C
void insertionSort(int arr[], int n) { 
    for(int i = 0; i < n-1; i++) {
        int end = i;
        int tmp = arr[end + 1];
        while(end >= 0) {
            if(tmp < arr[end]) {
                arr[end + 1] = arr[end];
                --end;
            } else break;
        }
        arr[end + 1] = tmp;
    }
}
```

### 为什么 `arr[end + 1] = tmp;` 要写在循环外面？

如果这个数比任何数都要小，那么当 `end--` 的时候，`end` 会 `<0`，跳出循环导致数据丢失。所以必须写在循环外面。
