# 非递归快排

## 核心思想

递归快排的本质是：**分区后，把左右子区间的信息压入栈，下次循环处理。**
非递归实现就是用显式栈模拟递归调用栈。

```C
void QuickSort(vector<int>& a, int left, int right) {
    stack<int> st;
    st.push(left);
    st.push(right);

    while (!st.empty()) {
        int r = st.top(); st.pop();
        int l = st.top(); st.pop();
        if (l >= r) continue;

        int key = a[l];
        int begin = l, end = r;
        while (begin < end) {
            while (begin < end && a[end] >= key) end--;
            while (begin < end && a[begin] <= key) begin++;
            if (begin < end) swap(a[begin], a[end]);
        }
        swap(a[l], a[begin]);

        st.push(l);
        st.push(begin - 1);
        st.push(begin + 1);
        st.push(r);
    }
}
```
