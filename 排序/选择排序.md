# 选择排序

核心思想：**每一轮从待排序区间中选出最小值和最大值，分别放到区间的开头和末尾。**

```C
void SelectionSort(int a[], int n) {
    int begin = 0, end = n - 1;
    while(begin < end) {
        int min = begin, max = begin;
        for(int i = begin + 1; i <= end; i++) {
            if(a[i] < a[min]) min = i;
            if(a[i] > a[max]) max = i;
        }
        swap(&a[min], &a[begin]);
        if(max == begin) max = min;
        swap(&a[max], &a[end]);
        begin++;
        end--;
    }
}
```

### `if(max == begin) max = min;` 的解释

假设数组 `[9, 1, 2, 5, 7]`：
- 交换前：`begin=0, min=1（值1）, max=0（值9）`
- 交换 begin 和 min：`[1, 9, 2, 5, 7]`
- `max` 还是 0，但 `a[0]` 现在是 1，不是最大值
- 所以需要更新 `max = min`
