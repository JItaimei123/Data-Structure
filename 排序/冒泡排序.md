# 冒泡排序

```C
void swap(int& a, int& b) {
    int tmp = a; a = b; b = tmp;
}

void bubbleSort(int arr[], int n) {
    for(int j = 0; j < n; j++) {
        for(int i = 0; i < n - 1 - j; i++) {
            if(arr[i + 1] < arr[i]) {
                swap(arr[i + 1], arr[i]);
            }
        }
    }
}
```
