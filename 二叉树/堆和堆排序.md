# 一、大堆和小堆

|堆类型|父子关系|特点|
|-|-|-|
|**大堆**|父亲 ≥ 孩子|**根是最大值**|
|**小堆**|父亲 ≤ 孩子|**根是最小值**|

堆的本质是一个完全二叉树。

## 小堆示例

```
        10
       /  \
     15    56
    /  \   /
   25  30 70
```

数组：`[10, 15, 56, 25, 30, 70]`

## 大堆示例

```
        70
       /  \
     56    30
    /  \   /
   25  10 15
```

数组：`[70, 56, 30, 25, 10, 15]`

# 二、调整算法

## 向上调整（小堆）

```C++
void AdjustUp(HPDataType* a, int child) {
    int parent = (child - 1) / 2;
    while (child > 0) {
        if (a[child] < a[parent]) {
            Swap(&a[child], &a[parent]);
            child = parent;
            parent = (child - 1) / 2;
        } else break;
    }
}
```

## 向下调整（小堆）

```C++
void AdjustDown(HPDataType* a, int n, int parent) {
    int child = parent * 2 + 1;
    while (child < n) {
        if (child + 1 < n && a[child + 1] < a[child]) child++;
        if (a[child] < a[parent]) {
            Swap(&a[child], &a[parent]);
            parent = child;
            child = parent * 2 + 1;
        } else break;
    }
}
```

## 堆的插入与删除

```C++
void HPPush(HP* php, HPDataType x) {
    php->a[php->size] = x;
    php->size++;
    AdjustUp(php->a, php->size - 1);
}

void HPPop(HP* php) {
    Swap(&php->a[0], &php->a[php->size - 1]);
    php->size--;
    AdjustDown(php->a, php->size, 0);
}

HPDataType HPTop(HP* php) {
    return php->a[0];
}
```

# 三、建堆

|建堆方式|遍历起点|时间复杂度|
|-|-|-|
|**向下调整建堆**|`(n-2)/2` → 0|**O(n)**|
|向上调整建堆|1 → n-1|O(n log n)|

一般用向下调整建堆。

```C++
// 向下调整建堆
for (int i = (n - 2) / 2; i >= 0; i--) {
    AdjustDown(a, n, i);
}
```

`(n-2)/2` 是最后一个非叶子节点。

# 四、小堆与大堆切换

|位置|小堆|大堆|
|-|-|-|
|找孩子|选较小的|选较大的|
|比较父子|孩子更小就交换|孩子更大就交换|

# 五、堆排序

|想要的结果|建什么堆|
|-|-|
|**升序**（从小到大）|**建大堆**|
|**降序**（从大到小）|**建小堆**|

## 升序（建大堆）

```C++
void HeapSort(int* a, int n) {
    for (int i = (n - 2) / 2; i >= 0; i--) AdjustDownBig(a, n, i);
    for (int i = n - 1; i > 0; i--) {
        Swap(&a[0], &a[i]);
        AdjustDownBig(a, i, 0);
    }
}
```

## 降序（建小堆）

```C++
void HeapSortDesc(int* a, int n) {
    for (int i = (n - 2) / 2; i >= 0; i--) AdjustDownSmall(a, n, i);
    for (int i = n - 1; i > 0; i--) {
        Swap(&a[0], &a[i]);
        AdjustDownSmall(a, i, 0);
    }
}
```

# 六、堆排序只需要向下调整

向上调整是给"动态插入"用的，堆排序只需要向下调整。

# 七、TopK问题

求文件中最大的前 K 个数，用小堆。

```C++
// 手写堆版本
vector<int> TopK(const string& filename, int k) {
    ifstream fin(filename);
    vector<int> heap(k);
    for (int i = 0; i < k && fin >> heap[i]; i++);
    BuildHeap(heap); // 建小堆
    int x;
    while (fin >> x) {
        if (x > heap[0]) { heap[0] = x; AdjustDown(heap, 0); }
    }
    return heap;
}

// priority_queue 快速实现
vector<int> TopKFromFile(const string& filename, int k) {
    ifstream fin(filename);
    priority_queue<int, vector<int>, greater<int>> pq;
    int x;
    while (fin >> x) { pq.push(x); if (pq.size() > k) pq.pop(); }
    vector<int> result;
    while (!pq.empty()) { result.push_back(pq.top()); pq.pop(); }
    return result;
}
```
