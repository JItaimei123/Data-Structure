# 二叉树性质总结

## 一、二叉树核心性质汇总（5条）

|序号|性质描述|公式|适用条件|说明|
|-|-|-|-|-|
|①|第 i 层最多节点数|**2^(i-1)**|任意二叉树（非空）|根节点为第 1 层，i ≥ 1|
|②|深度 h 的二叉树最大节点数|**2^h - 1**|满二叉树时取最大|所有节点数达到最大值|
|③|叶子节点数与度为2的节点数关系|**n0 = n2 + 1**|**所有二叉树**|度为0的节点（叶子） = 度为2的节点 + 1|
|④|满二叉树深度（由节点数反推）|**h = log₂(n+1)**|满二叉树|节点数为 n，每层全满|
|⑤|完全二叉树下标父子关系|**左：2i+1<br>右：2i+2<br>父：(i-1)/2**|完全二叉树（数组存储）|从 0 开始编号|

## 性质1：第 i 层最多有 `2^(i-1)` 个节点

- 第1层（根节点）：1 个节点 = `2^0`
- 第2层：最多 2 个节点 = `2^1`
- 第3层：最多 4 个节点 = `2^2`

每往下一层，每个节点最多可以产生 2 个孩子，所以下一层的节点数最多是上一层的 2 倍。

## 性质2：深度为 h 的二叉树，最多有 `2^h - 1` 个节点

深度 h = 1 → 最多 1 = `2^1 - 1`
深度 h = 2 → 最多 3 = `2^2 - 1`
深度 h = 3 → 最多 7 = `2^3 - 1`

等比数列求和：`1 + 2 + 4 + ... + 2^(h-1) = 2^h - 1`

## 性质3：`n0 = n2 + 1`

设 n0=叶子数, n1=度为1的节点数, n2=度为2的节点数

总节点数：`n = n0 + n1 + n2`
总边数：`n - 1 = n1 + 2*n2`

联立得：`n0 = n2 + 1`

## 性质4：满二叉树的深度 `h = log₂(n+1)`

满二叉树 `n = 2^h - 1` → `h = log₂(n + 1)`

## 性质5：完全二叉树下标父子关系

- 节点 i 的左孩子下标：`2*i + 1`
- 节点 i 的右孩子下标：`2*i + 2`
- 节点 i 的父节点下标：`(i-1)/2`

## 总结

|性质|一句话解释|
|-|-|
|①|每层最多翻倍|
|②|满树节点总数（等比数列求和）|
|③|叶子比二度节点多 1|
|④|满树深度反推|
|⑤|数组下标找父子|

## 补充说明

|编号|关键点|
|-|-|
|①、②|基于"根节点层数为 1"的前提|
|③|**适用所有二叉树**|
|④|仅适用于**满二叉树**|
|⑤|仅适用于**完全二叉树**（顺序存储）|

## 例题

### 例题1：已知 n2，求 n0
399个节点，199个度为2，叶子节点数？
`n0 = n2 + 1 = 199 + 1 = 200`

### 例题2：不适合顺序存储的结构
A 非完全二叉树 ❌ **不适合**
B 堆 ✅ 适合
C 队列 ✅ 适合
D 栈 ✅ 适合

**答案：非完全二叉树**

### 例题3：完全二叉树中，节点数为 2n，叶子数是多少？
偶数 → n1 = 1
`n0 + 1 + (n0 - 1) = 2n` → `n0 = n`

|总节点数 n|n1|叶子数 n0|
|-|-|-|
|奇数|0|`(n+1)/2`|
|偶数|1|`n/2`|

### 例题4：完全二叉树 531 个节点，高度是多少？
- h=9：前8层 255，总最多 511 → 531 > 511 ❌
- h=10：前9层 511，总最多 1023 → 531 ∈ [512, 1023] ✅

**答案：10**
