## 二、具体学习内容及重点

### 1. stack（栈）—— 重点

- **使用**：`push`、`pop`、`top`、`empty`、`size`
- **常见应用**：
  - 括号匹配
  - 逆波兰表达式求值
  - 单调栈（找下一个更大元素）
- **模拟实现**（重点）：
  - 基于 `vector` 或 `deque` 实现
  - 注意 `top` 和 `pop` 的顺序
  - 扩容策略（动态数组）

### 2. queue（队列）—— 重点

- **使用**：`push`、`pop`、`front`、`back`、`empty`、`size`
- **常见应用**：
  - BFS（广度优先搜索）
  - 滑动窗口最大值（配合 deque）
- **模拟实现**：
  - 基于 `list` 或 `deque` 实现
  - 注意头尾指针的维护

### 3. priority_queue（优先队列）—— 重点

- **使用**：
  - 默认大顶堆：`priority_queue<int> pq;`
  - 小顶堆：`priority_queue<int, vector<int>, greater<int>> pq;`
  - 自定义类型需重载 `<` 或传入比较函数
- **常见应用**：
  - TopK 问题（求前 K 大/小元素）
  - Dijkstra 算法优化
  - 合并 K 个有序链表
- **模拟实现**（了解即可）：
  - 基于堆（数组）实现 `push` 和 `pop`
  - 理解向上调整、向下调整

### 4. 容器适配器与 deque（了解即可）

- **概念**：stack/queue 是容器适配器，底层默认用 `deque`
- **deque**：双端队列，支持头尾 O(1) 插入删除，但随机访问较慢
- **为什么选 deque**：扩容不需要拷贝元素，比 vector 适合做 stack/queue 底层
- **你只需要知道这些**，不需要自己实现 deque

## 三、学习建议（按图片顺序）

1. **stack**：花 2 小时完成"介绍+使用+模拟实现"。手写一遍动态数组栈。
2. **queue**：花 2 小时完成"介绍+使用+模拟实现"。手写链式队列。
3. **priority_queue**：花 3 小时完成"介绍+使用+OJ练习"，再花 1 小时看懂模拟实现（不要求自己写）。
4. **容器适配器 + deque**：快速浏览 1 小时，理解概念即可，不必深究。

**总共约 8-9 小时**，你可以分 3-4 天完成。
